# CashSlip
Simple Expenditure Cash Slip system
Used technologies: JSTL, EL, MySQL DBMS, JDBC
Used IDE: Eclipse Mars
Deployed on Tomcat v.7: http://demo-expend.rhcloud.com/expend (-u root -p root)
